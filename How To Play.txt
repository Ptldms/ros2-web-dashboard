[ 초기 환경 구성 ]
1. Frontend (~/workspace/web_dashboard/frontend)
npm install

2. Backend (~/workspace/web_dashboard/backend)
npm install

3. ros2_bridge (~/workspace/web_dashboard/ros2_bridge)
colcon build
source install/setup.bash

--------------------------------------------------------------

[ 실행 방법 (bag ver) ]
1. bag 파일 실행
ros2 bag play 4planning_nocheon_250822/ -l  (예시)

2. react 실행 (~/workspace/web_dashboard/frontend)
npm start

3. 서버 실행 (~/workspace/web_dashboard/backend)
node server.js

4. ros2_bridge 노드 실행 (~/workspace/web_dashboard/ros2_bridge)
ros2 run ros2_bridge ws_node

--------------------------------------------------------------

[ 실행 방법 (실시간 ver) ]
1. 토픽 발행 확인
  a. /cmd/speed         (std_msgs/msg/Float32)
  b. /current_speed     (std_msgs/msg/Float32)
  c. /cmd/steer         (std_msgs/msg/Float32)
  d. /odometry/filtered (nav_msgs/msg/Odometry)
  e. /imu/processed     (sensor_msgs/msg/Imu)

2. react 실행 (~/workspace/web_dashboard/frontend)
npm start

3. 서버 실행 (~/workspace/web_dashboard/backend)
node server.js

4. ros2_bridge 노드 실행 (~/workspace/web_dashboard/ros2_bridge)
ros2 run ros2_bridge ws_node
